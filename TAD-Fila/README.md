# TAD Fila
*Código da fila desenvolvido em linguagem Java.*

### Principais operações:

* **enfileirar(object):** insere um elemento no fim da fila object ;
* **desenfileirar():** remove e retorna o elemento do início da fila;


### Operações auxiliares:

* **object inicio():** retorna o elemento do início sem removê-lo;
* **integer tamanho():** retorna o número de elementos armazenados;
* **boolean estaVazia():** indica se há elementos na fila.
